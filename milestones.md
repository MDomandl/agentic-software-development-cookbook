# Milestones

Dieses Dokument beschreibt den Arbeitsstand des Cookbooks „Agentic Software Development“.

## M0 – Projektstart

Status: erledigt

Ergebnis:

* ChatGPT-Projekt als Werkbank festgelegt
* GitHub-Repo als spätere Referenz festgelegt
* Praxisprojekte als Erfahrungsbasis festgelegt
* Grundstruktur des Repos skizziert

## M1 – Initiale Repo-Struktur

Status: erledigt

Ergebnis:

* privates GitHub-Repo erstellt
* erste Struktur angelegt
* `README.md` erstellt
* `menu.md` erstellt
* erstes Recipe angelegt

Enthaltene Dateien:

* `README.md`
* `menu.md`
* `recipes/asd-001-current-behavior-first.md`

## M2 – Projektsteuerung ergänzen

Status: in Arbeit

Ziel:

* offene Fragen sichtbar machen
* Meilensteine dokumentieren
* Cookbook-Entwicklung nachvollziehbar halten

Geplante Dateien:

* `open-questions.md`
* `milestones.md`

## M3 – Erste Recipe-Gruppe

Status: offen

Ziel:

Eine kleine erste Gruppe von Kernrezepten aufbauen.

Mögliche Recipes:

* `ASD-001 – Current Behavior First`
* `ASD-002 – Small Safe Steps`
* `ASD-003 – Separate Refactoring from Behavior Change`
* `ASD-004 – Agent Prompt as Work Order`
* `ASD-005 – Human Review Checkpoint`

## M4 – Erste Praxisvalidierung

Status: offen

Ziel:

Prüfen, welche Recipes sich in echten Projekten mehrfach bewähren.

Mögliche Erfahrungsquellen:

* `aktien_oop`
* `achtsam_morden`
* weitere kleinere Entwicklungsaufgaben

Ergebnis soll sein:

* Recipes nachschärfen
* unklare Recipes streichen oder umbenennen
* Anti-Patterns ergänzen
* Beispiele abstrahieren

## M5 – Erste veröffentlichbare Version

Status: offen

Ziel:

Eine kleine, runde Version des Cookbooks erstellen.

Möglicher Umfang:

* README
* Menü
* 5 bis 8 gute Recipes
* offene Fragen
* Meilensteine
* klare Abgrenzung des Projekts

Nicht-Ziel:

* vollständige Theorie agentischer Softwareentwicklung
* umfassendes Framework
* perfekte Taxonomie
